# 🔍 AI Search Algorithm Visualizer

## 📌 Overview
This project demonstrates the working of fundamental **Search Algorithms in Artificial Intelligence** through a simple pathfinding application. It helps in understanding how different algorithms explore a problem space to find a solution efficiently.

---

## 🎯 Objectives
- Implement core AI search algorithms  
- Compare uninformed and informed search techniques  
- Analyze efficiency and optimality of each algorithm  
- Provide a simple application-based demonstration  

---

## 🧠 Algorithms Implemented

### 🔹 Uninformed Search
- **Breadth First Search (BFS)**  
  Explores nodes level by level and guarantees the shortest path in unweighted graphs.

- **Depth First Search (DFS)**  
  Explores as deep as possible before backtracking; faster but not always optimal.

### 🔹 Informed Search
- **A* Algorithm**  
  Uses heuristics to find the most efficient path and is both complete and optimal.
